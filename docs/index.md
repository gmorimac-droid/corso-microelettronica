# Corso di Microelettronica Digitale

Benvenuto nel repository del corso.

## Obiettivi
Questo corso mette insieme:
- design RTL
- verifica funzionale
- uso di tool FPGA e ASIC
- script TCL
- esempi confrontati in VHDL, Verilog e SystemVerilog

## Come usare il corso
1. Parti da **00 Introduzione**
2. Segui i moduli in ordine
3. Apri gli esempi nella cartella `code/`
4. Aggiungi esercizi, immagini e note man mano

## Struttura didattica
Ogni modulo contiene:
- teoria
- esempi
- codice
- esercizi
- note di tool flow
